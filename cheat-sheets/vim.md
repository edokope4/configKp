## Comandos uso común en vim

| Comando | Descripción |
|----|----|
| :w | Guardar las modificaciones al archivo |
| :q | Salir de la edición del archivo |
| :x | Guardar y cerrar el archivo que se está editando | 
| :<nro_linea> | Salta hacia el número de linea especificado |
| /palabra | Busca "palabra" dentro del documento a editarse, con "n" avanza al próximo match y con ? se retrocede |
| :set number | Muestra el número de linea en el código |
| :tabnew | Abre una nueva pestaña para edición simultánea de archivos. |
| :tabe . | Abre una nueva pestaña y muestra el listao de archivos para abrir este en una nueva pestaña | 
| gt | Se mueve hacia la pestaña de la derecha |
| gT | Se mueve hacia la pestaña de la izquierda |
