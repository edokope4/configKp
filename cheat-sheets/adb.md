#### Comandos Adb: Android Debug Bridge


| Comando | Descripción | Ejemplo |
|----|----|----|
| adb devices | Entrega un listado de los dispositivos virtuales y físicos conectados | adb devices -l |
| adb start-server | Inicia servidor ADB | |
| adb kill-server | Termina el servidor ADB, suele ayudar cuando no responden los emuladores adecuadamente | |
