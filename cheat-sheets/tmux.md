## tmux Cheat Sheet

### Sesion
| Secuencia / comando | Descripción |
|----|----|
| tmux | Iniciar sesión. |
| tmux ls | Lista las sesionse activas. |
| tmux new -s main | Inicia una nueva sesión con un nombre específico (ej: main). |
| tmux attach -t <nombre/ID> | Entra (attach) a una sesión existente. |
| Ctrl+b + ? | Muestra la ayuda. | 
| Ctrl+b + : | Entra en el modo de comandos de tmux. |
| Ctrl+b + d | Manda la sesion a segundo plano (detach). |


### Paneles
| Secuencia / comando | Descripción |
|----|----|
| Ctrl+b + q | Muestra lista de paneles para elegir cuál usar. |
| Ctrl+b + z | Zoom. Pone la ventana actual pantalla completa, se vuelve de la misma manera. |

### Window
| Secuencia / comando | Descripción |
|----|----|
| Ctrl+b + c | Crea una nueva ventana |
| Ctrl+b + , | Cambia el nombre a la ventana |
| Ctrl+b + w | Lista las ventanas para elegir cuál usar. |
| Ctrl+b + p | Ventana Anterior. |
| Ctrl+b + n | Ventana próxima. |
| Ctrl+b + [0...9] | Cambia de ventana segun el número. |

### Varios
| Secuencia / comando | Descripción |
|----|----|
| Ctrl+b + t | Reloj. |
