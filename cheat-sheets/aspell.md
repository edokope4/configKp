| Comando | Descripción | Ejemplo |
|----|----|----|
| aspell check ARCHIVO | Revisar ortografia en ARCHIVO | aspell check texto.md |
| aspell --lang=LNG check ARCHIVO | Revisa ortografía de ARCHIVO con el idioma LNG | aspell --lang=es check texto.md |
| aspell -c ARCHIVO --mode=<modo> | Revisa ortografía, ignorando etiquetas propias del modo: html, tex, etc | aspell -c archivo.html --mode=html |
