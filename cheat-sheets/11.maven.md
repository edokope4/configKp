#### Comandos Maven

| Comando | Descripción |
|----|----|
| mvn clean | Limpieza para compilar bien |
| mvn install | Instala repo local con test |
| mvn package | Genera Jar con pruebas | 
| mvn test | Ejecuta las pruebas |
| mvn package -DskipTests | Compila lo test, pero no los ejecuta |
| mvn package -Dmaven.test.skip=true | Ignora completamente los test |
